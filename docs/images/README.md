# Allure Report Screenshot

Please add your Allure report screenshot here:

1. Run tests: `pytest tests/ -v --alluredir=allure-results`
2. Generate report: `allure serve allure-results`
3. Take screenshot of the report overview page
4. Save as `allure-report-overview.png` in this directory

## What to capture:

- Overview dashboard with test statistics
- Pass/Fail pie chart
- Execution timeline
- Test suites breakdown
- Severity distribution

This screenshot will be displayed in the main README.md file.
